## üí° Pro Tips & Tricks

### Getting Best Recommendations

1. **Be honest about budget** - Don't set it too low or too high
2. **Select multiple usage types** - Most people do more than one thing
3. **Consider future needs** - You'll have this laptop for years
4. **Don't over-spec** - More RAM/storage than needed wastes money

### Smart Shopping

1. **Compare prices** across retailers (Lazada, Shopee, Harvey Norman)
2. **Check warranty** - Official vs parallel imports
3. **Read reviews** - Check Lowyat forum for Malaysian experiences
4. **Time your purchase** - Sales during 11.11, 12.12, Chinese New Year
5. **Test in store** - If possible, feel the keyboard and screen

### Understanding Scores

**Composite Score (0-100) factors:**
- Performance (40%) - CPU, RAM, GPU
- Value for money (30%) - Price vs specs
- Build quality (15%) - Materials, durability
- Features (10%) - Ports, battery, screen
- Reliability (5%) - Brand track record

**A score of:**
- 90-100: Excellent overall package
- 80-89: Very good choice
- 70-79: Good for the price
- 60-69: Acceptable with tradeoffs
- <60: Consider alternatives

### Keyboard Shortcuts

- `Ctrl/Cmd + K`: Focus search
- `Esc`: Close modals
- `Ctrl/Cmd + Enter`: Send chat message
- `Ctrl/Cmd + F`: Find on page

## üÜò Troubleshooting

### "Can't find any laptops"

- Check your filters (may be too restrictive)
- Clear all filters and try again
- Try broadening your budget range

### "AI not responding"

- Check internet connection
- Refresh the page
- Clear browser cache
- Try in incognito/private mode

### "Image upload not working"

- Check file size (<5MB)
- Use JPEG, PNG, or WebP format
- Try different photo
- Check camera permissions

### "Can't create account"

- Check email format
- Password must be 8+ characters with letters and numbers
- Check spam folder for verification email
- Try magic link instead
