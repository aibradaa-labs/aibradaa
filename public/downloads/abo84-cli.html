ABO-84 CLI Tool

Command-line interface for CI/CD integration and terminal workflows

Requirements: Node.js 18+ | npm or yarn | Ultimate tier subscription

Installation

1

Install via npm

npm install -g @aibradaa/abo84-beta

Or with yarn:

yarn global add @aibradaa/abo84-beta
2

Authenticate

abo84 login

Follow the prompts to sign in with your AI Bradaa account. Your Ultimate tier subscription will be verified.

3

Verify Installation

abo84 --version

# Output:
# ABO-84 v1.0.0-beta
# AI Bradaa Observer - 84

Command Reference

abo84 analyze [path]

Analyze code in specified directory or file

abo84 analyze ./src
abo84 analyze ./src/index.js
abo84 analyze . --recursive

abo84 scan --security

Run security-focused scan (OWASP, CWE vulnerabilities)

abo84 scan --security ./src
abo84 scan --security --severity=critical

abo84 fix --auto

Automatically fix detected issues (safe transformations only)

abo84 fix --auto ./src
abo84 fix --auto --severity=high
abo84 fix --auto --preview # Preview changes first

abo84 report --format=[type]

Generate analysis report in various formats

abo84 report --format=json > report.json
abo84 report --format=html > report.html
abo84 report --format=markdown > REPORT.md

abo84 watch [path]

Continuous analysis mode (watches for file changes)

abo84 watch ./src
abo84 watch ./src --auto-fix

abo84 config

Configure CLI settings and preferences

abo84 config set severity high
abo84 config set autoFix true
abo84 config list

CI/CD Integration

GitHub Actions

name: ABO-84 Code Analysis

on: [push, pull_request]

jobs:
analyze:
runs-on: ubuntu-latest
steps:
- uses: actions/checkout@v3
- uses: actions/setup-node@v3
with:
node-version: '18'
- name: Install ABO-84
run: npm install -g @aibradaa/abo84-beta
- name: Authenticate
run: abo84 login --token ${{ secrets.ABO84_TOKEN }}
- name: Run Analysis
run: abo84 analyze ./src --fail-on critical
- name: Generate Report
run: abo84 report --format=json > analysis.json
- name: Upload Report
uses: actions/upload-artifact@v3
with:
name: abo84-report
path: analysis.json

GitLab CI

abo84_analysis:
image: node:18
script:
- npm install -g @aibradaa/abo84-beta
- abo84 login --token $ABO84_TOKEN
- abo84 analyze ./src --fail-on critical
artifacts:
reports:
codequality: analysis.json

Pre-commit Hook

Add to .git/hooks/pre-commit:

#!/bin/bash

# Run ABO-84 analysis on staged files
abo84 analyze $(git diff --cached --name-only --diff-filter=ACM | grep -E '\.(js|ts)$') --fail-on high

if [ $? -ne 0 ]; then
echo "❌ ABO-84 found critical issues. Commit blocked."
exit 1
fi

echo "✅ ABO-84 analysis passed"
exit 0

Configuration File

Create .abo84rc.json in your project root:

{
"version": "1.0.0",
"severity": "high",
"autoFix": true,
"failOnSeverity": "critical",
"exclude": [
"node_modules/**",
"dist/**",
"*.test.js"
],
"rules": {
"SEC-001": "error",
"PERF-042": "warn",
"QUAL-015": "off"
},
"output": {
"format": "json",
"file": "./reports/abo84.json"
}
}
