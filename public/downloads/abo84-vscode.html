ABO-84 VS Code Extension

Real-time AI-powered code analysis integrated directly into Visual Studio Code

⚡ Install Extension View in Marketplace
Requirements: Visual Studio Code 1.80+ | Ultimate tier subscription | Node.js 18+ (optional for offline mode)

Installation Methods

1

Method 1: VS Code UI (Recommended)

  1. Open Visual Studio Code
  2. Click Extensions icon (Ctrl+Shift+X / Cmd+Shift+X)
  3. Search for "ABO-84"
  4. Click "Install" on the ABO-84 extension by AI Bradaa
  5. Reload VS Code when prompted
2

Method 2: Command Line

code --install-extension aibradaa.abo84-beta

This installs the extension globally for all VS Code workspaces.

3

Authenticate & Activate

After installation, authenticate with your AI Bradaa account:

  1. Open Command Palette (Ctrl+Shift+P / Cmd+Shift+P)
  2. Type: ABO-84: Sign In
  3. Enter your AI Bradaa credentials
  4. Extension will verify your Ultimate tier subscription
4

Start Analyzing

Open any JavaScript/TypeScript project. ABO-84 will automatically start analyzing:

  • Issues appear in the Problems panel
  • Inline diagnostics with hover details
  • Quick fixes available via lightbulb icon
  • Status bar shows code quality score

Features

Real-time Analysis

Get instant feedback as you type. Issues are detected and highlighted in real-time with detailed explanations.

Auto-fix Quick Actions

One-click fixes for common issues. Click the lightbulb or press Ctrl+. (Cmd+.) to apply automatic fixes.

Code Quality Score

See your overall code quality score in the status bar. Click to view detailed breakdown by category.

Security Scanning

Detects OWASP vulnerabilities, injection risks, and unsafe patterns. Critical issues are highlighted immediately.

Performance Insights

Identifies performance bottlenecks, memory leaks, and inefficient algorithms with optimization suggestions.

Custom Configuration

Configure severity thresholds, enable/disable rules, and customize analysis behavior per project.

Configuration

Workspace Settings

Create .vscode/settings.json in your project:

{
"abo84.enabled": true,
"abo84.severity": "high", // "critical" | "high" | "medium" | "low"
"abo84.autoFix": true,
"abo84.excludePatterns": ["node_modules/**", "dist/**"],
"abo84.offlineMode": false
}

Project Configuration

Create .abo84rc.json in project root:

{
"rules": {
"SEC-001": "error",
"PERF-042": "warn",
"QUAL-015": "off"
},
"ignore": ["*.test.js", "*.spec.ts"]
}
